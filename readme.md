


### Parse File
A function that can be used to parse targets from a file.
Accepeted hosts include:
- IP address 
- CIRD Notation (IPv4 only at the moment)
- FQDN 

#### Example

File1.txt contents:
```
host1.light.local
192.168.10.0/28
10.10.10.10
10.10.20.10
10.20.30.40
lampsforsale.light
```
